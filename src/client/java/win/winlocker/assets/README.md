# WinLocker Logo

This directory contains the logo file for WinLocker client.

## Logo Information
- File: logo.png
- Size: 64x64 pixels
- Format: PNG with transparency
- Location: src/client/java/win/winlocker/assets/

## Logo Design
The logo represents "WV" (WinLocker Visuals) with:
- Blue "W" letter
- White "V" letter
- Decorative underline
- Clean, modern design

## Usage
The logo is rendered in the WaterMark at the beginning of the HUD display.
